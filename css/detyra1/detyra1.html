Shembull CSS

Kjo është një faqe shembull që përdor kodin CSS të dhënë.

CSS (Cascading Style Sheets) është një gjuhë e përdorur për të stiluar dhe paraqitur dokumente të shkruara në HTML.

Mund të përdoret për të përcaktuar ngjyrat, fontet, spacing, dhe shumë karakteristika të tjera të prezantimit që shfaqen në një faqe web.
